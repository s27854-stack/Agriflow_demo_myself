Setup Guide

  1. Install Libraries
    Open Arduino IDE → Tools → Manage Libraries → Install ESP32Servo and DHT sensor library (if using DHT sensor)
  2. Wire Your Sensors
    Moisture Sensor → GPIO 34, Servo → GPIO 18, DHT22 → GPIO 4
  3. Flash & Open Portal
    Upload any .ino sketch to ESP32. On first boot, connect to WiFi SmartSprinkler-Setup (password: setup123)
  4. Configure WiFi & Server
    A setup page pops up automatically. Select your home WiFi, enter the server IP shown by the dashboard, then click Save
  5. You're Done!
    ESP32 reboots and connects automatically. No code editing needed!
To reconfigure: Hold the BOOT button for ~3 seconds while powered on. The portal reopens so you can enter new WiFi/server details.
